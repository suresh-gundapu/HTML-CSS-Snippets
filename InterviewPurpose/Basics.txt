page.

User Timing API: Provides programmers with high-precision timestamps for
measuring application performance.
Vibration API: Provides access to the device's vibration functionality.

<embed>: Used to embed external applications, usually multimedia content

<track>: Specifies text tracks for audio and video components.

HTML5 Semantic Tags :
=====================

Engine Optimization.
Semantic elements = elements with a meaning.
A semantic element clearly describes its meaning to both the browser and the developer.
Examples of non-semantic elements: <div> and <span> - Tells nothing about its content.
Examples of semantic elements: <form>, <table>, and <article> - Clearly defines its content.
New Semantic Elements
Semantic elements : header, nav, section, article, aside, and footer
Form controls
: calendar, date, time, email, url and search
Media tags
: audio, video, and canvas
